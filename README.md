# Real-time renderer

![](https://i.imgur.com/fFzDX3k.png)

## Features

- Metallic/roughness workflow
- Deferred rendering
- Normal mapping

![](https://i.imgur.com/7wl0QNT.png)

- Diffuse environment lighting

![](https://i.imgur.com/GDBCQPG.png)

- Anti-aliasing with temporal supersampling (TAA)
